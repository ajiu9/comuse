# Work with AI

## Agent Skills

The [Comuse Skills](https://github.com/ajiu9/comuse-skills) are AI Agent Skills maintained by the Comuse.

After installing the skill, when you use an AI Agent to assist with developing Vue, it can automatically leverage the rich feature set provided by Comuse.

This allows the agent to accurately use Comuse functions **without requiring an internet connection or additional permissions**.

### Installation

```bash
npx skills add comuse/skills
```
